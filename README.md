# McMaster Economics Study Hub

A personal study hub for my McMaster Economics courses — combining course outlines, textbook chapter content, data visualisations, practice problem sets, and links to relevant external resources, all in one place.

The goal is to make it genuinely easy to learn everything for each course without bouncing between PDFs, lecture slides, and random YouTube tabs.


## What's inside each lesson

Every chapter page is a self-contained, rendered HTML lesson that includes:

- **Full chapter content** — drawn from the textbook and course outline, written as a taught lesson rather than a summary
- **Worked examples** — step-by-step, matching the depth expected on assessments
- **Data visualisations** — graphs generated with Python/matplotlib and embedded directly in the page (no external dependencies)
- **Practice problems** — 5 problems per chapter with hidden, revealable answers; Problem 5 is always an applied economics question
- **External resource links** — Khan Academy, Professor Leonard, 3Blue1Brown, Paul's Online Math Notes, MIT OCW, and others, placed inline next to the content they support
- **Chapter summary table** — key concepts, formulas, and when to use them
- **Self-check checklist** — skills you should be able to do before moving on
