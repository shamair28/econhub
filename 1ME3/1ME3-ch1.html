Section 1

Why This Chapter Matters

Economics is fundamentally about relationships: how does price affect the quantity people want to buy? How does income affect spending? How do two firms' output decisions interact?

The simplest way to model any relationship between two variables is a straight line — a linear equation. Before you can do any serious economics math (optimization, equilibrium analysis, calculus), you need to be completely fluent with lines.

Big picture: Every supply curve, every demand curve, every budget constraint is, at its core, a line. Finding where two lines cross is how we find equilibrium — a calculation you will do hundreds of times in this course.
Section 2

Slope-Intercept Form: y = ax + b

This is the most useful form. Every line can be written as y = ax + b, where:

SymbolNameMeaning
aslopeFor every 1-unit increase in x, y changes by a
by-interceptThe value of y when x = 0

What slope actually means

Slope measures steepness and direction:

  • a > 0 → line goes up-right (positive relationship)
  • a < 0 → line goes down-right (negative relationship)
  • a = 0 → flat horizontal line (no relationship)
  • Large |a| → steep line; small |a| → shallow line

Intuition: If a = 3, then every time x increases by 1, y jumps by 3. If a = −0.5, every time x increases by 1, y drops by half.

Worked Example
A line has slope a = 2 and crosses the y-axis at b = −3. Write its equation and find the value of y when x = 5.

Equation: y = 2x − 3

At x = 5:  y = 2(5) − 3 = 10 − 3 = 7

So the point (5, 7) lies on this line.

Section 3

General Form: ux + vy = w

Sometimes a line is written symmetrically, without solving for y:

ux + vy = w

This form treats x and y equally — neither is "the output." It's especially useful for setting up systems of equations. To graph it or interpret it, convert to slope-intercept by solving for y.

Worked Example — Converting Forms
Convert 2x + 3y = 12 to slope-intercept form.
2x + 3y = 12
      3y = 12 − 2x       ← subtract 2x from both sides
       y = (12 − 2x) / 3  ← divide both sides by 3
       y = 4 − (2/3)x     ← distribute the division
       y = −(2/3)x + 4    ← rearrange to y = ax + b form

Result: slope a = −2/3, intercept b = 4

✓ Tip: After converting, always double-check by plugging in a simple value. At x = 0: y = 4. Check in the original: 2(0) + 3(4) = 12 ✓
Section 4

Finding a Line's Equation from Two Points

If you know any two points on a line, you can always reconstruct the full equation. This is "rise over run" — how much y changes divided by how much x changes.

Given points (x₀, y₀) and (x₁, y₁):

        y₁ − y₀
a  =  -----------
        x₁ − x₀

Once you have a, plug either point into y = ax + b and solve for b.

Worked Example
Find the equation of the line passing through (1, 3) and (4, 9).
Step 1 — Compute slope:
a = (9 − 3) / (4 − 1) = 6 / 3 = 2

Step 2 — Find intercept using point (1, 3):
y = ax + b
3 = 2(1) + b
b = 1

Result: y = 2x + 1

Check with (4, 9):  y = 2(4) + 1 = 9  ✓
⚠ Common mistake: Mixing up which coordinate is which. Always write (y₁ − y₀) / (x₁ − x₀) — y on top, x on bottom — and keep the same order (same point first in both numerator and denominator).
Section 5

Solving 2×2 Systems of Equations

A system of equations is two equations that must both be true at the same time. Geometrically, each equation is a line, and the solution is the single point where both lines cross.

Equation 1:  y = 2x + 1
Equation 2:  y = −x + 7

Method 1: Substitution

Idea: Since both equations say what y equals, set the right-hand sides equal to each other.

Worked Example — Substitution
2x + 1 = −x + 7    ← both expressions equal y
3x = 6              ← add x to both sides, subtract 1
 x = 2              ← divide by 3

Substitute x = 2 back into either equation:

y = 2(2) + 1 = 5

Solution: (x, y) = (2, 5)
Check in Equation 2: y = −2 + 7 = 5 ✓

Method 2: Elimination

Idea: Rewrite both equations in general form, then add (or subtract) them to cancel one variable.

Worked Example — Elimination
Equation 1:  2x − y = −1
Equation 2:   x + y =  7
─────────────────────────
Add both:    3x     =  6  ← y terms cancel
              x     =  2

Substitute into Eq. 2:  2 + y = 7  →  y = 5

Solution: (2, 5)

✓ When to use which method?
Use substitution when one equation is already solved for a variable.
Use elimination when both equations are in general form and a variable cancels easily.

Geometric Interpretation: Three Possible Outcomes

SituationGeometric meaningSolution count
Different slopesLines cross at one pointOne unique solution
Same slope, different interceptParallel lines — never meetNo solution
Same slope, same interceptIdentical lines — overlap completelyInfinitely many

Section 6

Economics Application: Supply, Demand & Equilibrium

This is where linear equations come alive in economics. The entire foundation of microeconomic market analysis rests on one idea: finding where two lines intersect.

The Setup

CurveEquationEconomic logic
DemandQᵈ = a − bP (a, b > 0)Higher price → less quantity demanded (downward slope)
SupplyQˢ = c + dP (d > 0)Higher price → more quantity supplied (upward slope)
EquilibriumSet Qˢ = Qᵈ, solve for P*Market clears — no surplus, no shortage
Worked Example — Market Equilibrium
Demand: Qᵈ = 100 − 4P, Supply: Qˢ = −20 + 6P. Find P* and Q*.
Step 1 — Set Qˢ = Qᵈ:
100 − 4P = −20 + 6P
     120 = 10P
      P* = 12

Step 2 — Find Q*:
Q* = 100 − 4(12) = 100 − 48 = 52

Verify with supply:  Q* = −20 + 6(12) = 52  ✓

Equilibrium: P* = 12, Q* = 52

Key insight: Finding market equilibrium is just solving a 2×2 system of linear equations — exactly the technique from Section 5. The "economics" and the "math" are the same operation.
Section 7

Practice Problems

Work through each problem before revealing the answer. Aim to solve Problems 1–4 in under 2 minutes each.

Problem 1 — Read a line

The line y = −3x + 7 has what slope and y-intercept? What is y when x = 2?

Show answer
Slope = −3, intercept = 7.
At x = 2: y = −3(2) + 7 = 1
Problem 2 — Convert from general form

Convert 4x − 2y = 10 to slope-intercept form.

Show answer 
4x − 2y = 10
    −2y = 10 − 4x
      y = 2x − 5
Slope = 2, intercept = −5
Problem 3 — Line from two points

Find the equation of the line through (2, 1) and (6, 9).

Show answer 
a = (9−1)/(6−2) = 8/4 = 2
Using (2,1): 1 = 2(2) + b → b = −3
Equation: y = 2x − 3
Problem 4 — Solve the system

3x + y = 11 and x − y = 1

Show answer 
  3x + y = 11
+  x − y =  1
──────────────
  4x     = 12  →  x = 3
Substitute: 3 − y = 1 → y = 2
Solution: (3, 2)
Problem 5 — Market equilibrium

Demand: Qᵈ = 80 − 2P. Supply: Qˢ = −10 + 3P. Find P* and Q*.

Show answer 
80 − 2P = −10 + 3P
      90 = 5P
      P* = 18

Q* = 80 − 2(18) = 44
Verify: −10 + 3(18) = 44  ✓
P* = 18, Q* = 44
